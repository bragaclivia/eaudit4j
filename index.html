eAudit4j

Yet another audit library / framework for Java. This one is processor driven, and also allows replacing the auditors for maximum flexibility.

eAudit4J Library

This audit library provides an extendable audit framework.

The following extension points are supported:

  • Custom processor implementations
  • Custom event implementations
  • Custom event attribute
  • Custom audit strategies (e.g. synchronous, asynchronous)
